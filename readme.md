[mkdocs](http://www.mkdocs.org/)

### 安装

```
$ python --version
Python 2.7.2

$ pip --version
pip 1.5.2

# 升级 pip
$ pip install --upgrade pip
# 安装 mkDocs
$ pip install mkdocs

$ mkdocs --version
mkdocs, version 0.15.3

$ mkdocs new my-project
$ cd my-project
$ mkdocs serve
```

* 实时编辑

`mkdocs serve` 

* 发布

`mkdocs build`, `mkdocs build --clean`
